# AI Brand Creative Generator

## Project Goal

This project demonstrates a structured prompt workflow for generating brand-aligned marketing creatives using generative AI models.

The objective is to translate a creative brief into a repeatable AI workflow that generates consistent brand visuals and messaging.

## Creative Brief

Design a Halloween promotional image for Midnight Coffee promoting a limited edition pumpkin latte.

Brand attributes:
- tone: premium and mysterious
- colours: black and gold
- style: minimalist luxury coffee aesthetic

## Workflow

Creative Brief  
↓  
Brand Analysis Prompt  
↓  
Messaging Generation  
↓  
Image Prompt Generation  
↓  
Image Model Playground AI  
↓  
Generated Creative Output

## Tools Used

- LLM: GPT / Gemini for prompt structuring
- Image Model: Playground AI
- Documentation: Markdown + GitHub

## Prompt Architecture

The prompts are structured into modular components:

1. Brand analysis
2. Message generation
3. Image prompt generation

This allows prompts to be reused across multiple campaigns.

## Evaluation

Generated outputs were evaluated on:

- brand alignment
- visual clarity
- creative consistency
